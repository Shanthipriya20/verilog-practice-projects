# Verilog Practice Projects

This repository contains digital design projects implemented using Verilog as part of my academic learning and hands-on practice in VLSI design.

## Projects Included
- FIFO (Synchronous & Asynchronous)
- UART (Universal Asynchronous Receiver Transmitter)
- Traffic Light Controller
- Vending Machine (FSM Based)
- Washing Machine Controller
- Parity Checker

## Tools Used
- xillinx
- quartus prime

## Note
These projects were developed through self-practice and peer discussions, with focus on understanding digital design concepts and simulation.
